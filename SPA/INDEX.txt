Q: Database error?
A: python manage.py migrate

Q: Quên mật khẩu admin?
A: python manage.py changepassword admin

A: Sửa SPA/static/css/auth.css
   Thay giá trị --primary-color

Q: Làm sao để gửi email?
A: Xem README_AUTH.md phần "Email Configuration"

Q: Còn cần làm gì?

   - Quên mật khẩu (Password reset)

📞 LIÊN HỆ & HỖ TRỢ
═════════════════════════════════════════════════════════════════════════════════

Nếu gặp vấn đề:

1. Kiểm tra Server:
   - Mở DevTools (F12)
   - Xem tab Console có lỗi không?
   - Xem tab Network có failed requests?

2. Kiểm tra Code:
   - cd vào thư mục SPA
   - python manage.py runserver
   - Xem output console có lỗi Python không?

3. Reset Database:
   - Xóa file db.sqlite3 (nếu cần)
   - python manage.py migrate

4. Clear Cache:
   - Browser: Ctrl+Shift+Delete
   - Python: Xóa __pycache__ folders

5. Đọc Tài Liệu:
   - Xem QUICK_START.md
   - Xem README_AUTH.md
   - Xem CHECKLIST.md
