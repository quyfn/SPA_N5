sau khi clone về thì mở project ra

tụi bây cứ code, lưu ý là phải nhớ bây code ra những file nào tránh nhầm file với đứa khác
ví dụ: t code xong, t push lên rồi t sẽ thông báo lại cho tụi bây
thì lúc đó tụi bây phải gõ lệnh:
git stash (để lưu tạm cái code tụi bây tự code nãy giờ)
git pull (tải code mới về, nó sẽ cập nhật code mới của t mới push lên)
git stash pop (paste cái code của tụi khi trước lưu tạm ấy)

làm xong rồi thì đẩy code lên github như bình thường
